DoGraf.atualiza_value_tick(numero, yloc, yticks)¶

Calculamos qual é o valor que escrevemos no topo da barra do plt.bar ou plt.hist

Parâmetros

  • numero (float) – numero a transformar

  • yloc (list of float) – localização dos ticks do axis-y

  • yticks (list of float) – valores dos ticks do axis-y

Retorno

novo valor para o numero

DoGraf.custom(c, folder_name, dirFiles, dirImagens, grafs)¶

Função vai ser chamada para cada diretoria que existe no .conf . Vai verificar que tipo de graficos temos para criar e chama a função customAuxiliar

Parâmetros

  • c (dict) – secçao do .conf que corresponde uma das nossas diretorias que tem informaçao para gerar graficos. Ex: c = {'Hist': {'separa': 'True', 'Ficheiros': {'foo1': {'bins': 'range(4)','Specs': {}}, 'foo2': {'Specs': {'bins': ['range(4', '12)']}}, 'foo3':etc

  • folder_name (str) – Nome do diretoria que vamos fazer os graficos Ex: folder_name = barrasTeste.2

  • dirFiles (str) – Path para a diretoria do ficheiros que tem valores dos graficos

  • dirImages (str) – Path para a diretoria onde as pastas dos graficos vao ser guardados

  • grafs (list of str) – lista que contem os tipos de graficos que conseguimos criar Ex: grafs = ["Linhas", "Barras", "Hist"]

DoGraf.customAuxiliar(files, tipoGrafico, c, folder_name, dirFiles, dirImagens, grafs)¶

vai averiguar se foi especificado fazer plot dos graficos todos ou só de alguns e se estes vao aparecer na mesma plt.figura() ou em plt.figura() diferentes. Chamamos assim uma destas funçoes ficheiros_separados(), ficheiros_juntos(), tudo_separado(), tudo_juntos()

Parâmetros

  • folder_name (str) – Nome do diretoria que vamos fazer os graficos Ex: folder_name = barrasTeste.2

  • dirFiles (str) – Path para a diretoria do ficheiros que tem valores dos graficos

  • dirImages (str) – Path para a diretoria onde as pastas dos graficos vao ser guardados

  • grafs (list of str) – lista que contem os tipos de graficos que conseguimos criar Ex: grafs = ["Linhas", "Barras", "Hist"]

DoGraf.desenho(tipoGrafico, x, dicio, confTipoGrafico, i, numFiles, numPilares, especificacoes, fileName, minY, maxY, bar_empilha)¶

Vai tratar dos aspetos do «desenho» do grafico (cor, edgecolor, valores no topo das Barras, Hist-bins) ; tambem colocamos aqui os yticks para ficarmos com os valores que vao aparecer no y-axis e podermos ajustar os valores que poderam aparecer no topo das tabelas dos plt.bar e plt.hist()

Parâmetros

  • tipoGrafico (str) – ajudanos a saber que tipo de grafico vamos criar. Ex: `` tipoGrafico = «Linhas»``

  • x (np array) – array com os valores que formam os graficos

  • dicio (dict) – especificacoes para o grafico (que normalmente entram em plt.plot()/bar()/hist() ) Ex: dicio = {'color': 'g', 'linestyle': 'None', 'marker': 'h', 'label': 'teste legenda2'}

  • confTipoGrafico (dict) – secçao do .conf que corresponde uma das nossas diretorias que tem informaçao para gerar graficos. Ex: confTipoGrafico = {'separa': 'True', 'Ficheiros': {'foo1': {'bins': 'range(4)', 'Specs': {}}, 'foo2': {'Specs': {'bins': ['range(4', '12)']}}, 'foo3': {'Specs': {}}}, 'Specs': {'value': {'precisao': '3', 'fontsize': '7'}, 'legend': {'loc': 'upper left'}}}

  • i (int) – indice do grafico de entre de todos aqueles que selecionamos para fazer plot

  • numFiles (int) – numero do ficheiro de valores de entre de todos aqueles que selecionamos para fazer plot

  • numPilares (int) – em plt.bar() é o numero de barras que aparecem numa figura. Default é 4

  • especificacoes (dict) – especificações que vamos usar usar no grafico

  • fileName (str) – nome do ficheiro de valores, vamos, possivelmente, usar como nome da plt.figure() que vamos guardar

  • minY (float) – menor valor dos y de entre todos os ficheiros que vamos fazer plot dentro de uma diretoria

  • maxY (float) – maior valor do y de entre todos os ficheiros que vamos fazer plot dentro de uma diretoria

  • bar_empilha (list of list of int) – cada linha representa um ficheiro; cada coluna representa uma barra do plot.bar(); os valores é o y começamos a fazer plot das barras de um ficheiro

Retorno

numero de figuras os fizemos plot <=> numero de figuras que tem de ser guardadas

Retorno

-1 caso existam erros

  • dirFiles (str) – Path para a diretoria do ficheiros que tem valores dos graficos

  • confTipoGrafico (dict) – secçao do .conf que corresponde uma das nossas diretorias que tem informaçao para gerar graficos. Ex: confTipoGrafico = {'separa': 'True', 'Ficheiros': {'foo1': {'bins': 'range(4)', 'Specs': {}}, 'foo2': {'Specs': {'bins': ['range(4', '12)']}}, 'foo3': {'Specs': {}}}, 'Specs': {'value': {'precisao': '3', 'fontsize': '7'}, 'legend': {'loc': 'upper left'}}}

  • numPilares (int) – em plt.bar() é o numero de barras que aparecem numa figura. Default é 4

  • minY (float) – menor valor dos y de entre todos os ficheiros que vamos fazer plot dentro de uma diretoria

  • maxY (float) – maior valor do y de entre todos os ficheiros que vamos fazer plot dentro de uma diretoria

  • Freqmax (int) – valor que se repete mais vezes em um ficheiro de entre todos os ficheiros que vamos fazer plot dentro de uma diretoria

  • files (list) – lista dos ficheiros com valores para criar os graficos. Ex: files = ['timenewformulation2.txt', 'timenewformulation0.txt', 'timenewformulation1.txt']

Retorno

-1 caso existam erros, 0 caso contrario

DoGraf.unirSpecs(especificacoes, confTipoGrafico)¶

Une as Specs especificas do ficheiro com as Gerais (As especificações dos ficheiros tem + prioridade) «especificacoes» sao as Specs individuais dos files

Parâmetros

  • especificacoes (dict) – especificações individuais dos ficheiros

  • confTipoGrafico (dict) – secçao do .conf que corresponde uma das nossas diretorias que tem informaçao para gerar graficos. Ex: confTipoGrafico = {'separa': 'True', 'Ficheiros': {'foo1': {'bins': 'range(4)', 'Specs': {}}, 'foo2': {'Specs': {'bins': ['range(4', '12)']}}, 'foo3': {'Specs': {}}}, 'Specs': {'value': {'precisao': '3', 'fontsize': '7'}, 'legend': {'loc': 'upper left'}}}

Retorno

especificacoes, caso nao existam Specs em confTipoGrafico. aux, sendo aux uma copia das Specs Gerais que foi atualizada com as especificações do grafico
